How the engine works

Each conversion category defines a base unit and a mapping of unit factors relative to that base. A conversion is a two-step operation: normalize the value into the base unit, then divide by the target unit's factor. Temperature and fuel economy use dedicated formulas because they aren't purely linear.

Conventions

  • Data storage uses the decimal (SI) convention: 1 KB = 1,000 B.
  • Time uses the Julian year (365.25 days) and average Gregorian month.
  • Fluid volumes (cup, pint, quart, gallon) use US customary values.

Tech stack

FastAPI, Jinja2 templates, HTMX, Tailwind CSS, SQLAlchemy, and Pydantic. The codebase is intentionally modular — converters, services, repositories, and routers are split into small focused modules.
